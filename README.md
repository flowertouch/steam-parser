Steam Profile Enhancer (pars) 
pars — это мощный UserScript для Tampermonkey, предназначенный для расширения функционала профилей Steam, сфокусированный на игроках CS2. Он добавляет информативный виджет с данными, которые обычно скрыты или требуют перехода на сторонние ресурсы.

Основные возможности:
Детектор блокировок: Распознает активные Community Ban (КТ), VAC-баны и скрытые ограничения, выделяя их статус цветом.

График опыта (XP Chart): Интеграция с сервисом csxp.gg для отображения динамики получения опыта в CS2 через наглядный SVG-график.

Генератор Friend Code: Автоматическое вычисление кода друга из SteamID64 для быстрого добавления в игру.

История банов: Отображение истории блокировок пользователя.

Коллекция значков (Pins/Medals): Удобный просмотр медалей и значков из инвентаря CS2 прямо в виджете.

Кастомизация интерфейса: * Выбор фонов (Anime, Japan) или прозрачного режима для виджета.

Двуязычный интерфейс (RU/EN).

Копирование SteamID и Friend Code в один клик.

Технический стек :
Language: JavaScript (ES6+)

Platform: Tampermonkey / Grease-monkey

Libraries: CryptoJS (для шифрования кодов).

API/Data: Интеграция с csxp.gg и steamcommunity.com.

 Установка :
Установите расширение Tampermonkey для вашего браузера.

Создайте новый скрипт и вставьте содержимое файла pars.user.js (или скопируйте код из этого репозитория).

Перейдите на любой профиль Steam, чтобы увидеть изменения.
